dump-python-log Command

Prints out the latest contents of the Python scripting error log to the command log.

Each entry represents a single Python scripting error which occurred during the map rendering. Each entry includes:

  • the time the error occurred,
  • the error description (message),
  • variables that were in the execution context at the time the error occurred.

Usage 

 dump-python-log
