zoom-map-scale Command

Contents

  • Usage
  • Map Scales And Display Size
  • Examples

Sets the map zoom level based on the map scale ratio.

Usage 

zoom-map-scale <scale>

Parameters:

  • scale: specifies the map scale ratio.

Map Scales And Display Size

The map scale depends on the actual physical size of pixels on your display device (monitor). Maperitive uses pixels-per-inch (PPI) units to represent the size of pixels. You can adjust the PPI by setting the display.ppi setting to the appropriate value.

PPI calculator: http://members.ping.de/~sven/dpi.html.

Examples 

 zoom-map-scale 50000

Sets the zoom level equivalent to map scale 1 : 50,000
