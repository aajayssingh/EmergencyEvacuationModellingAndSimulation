Maperitive 2.3.33 documentation

set-setting Command

Sets a value of a specific Maperitive user setting.

Usage 

 set-setting 
     name=<setting name>
     value=<setting value>

Parameters:

  • name: specifies the setting name.
  • value: specifies the new value for the setting.

Examples 

 set-setting name=web.proxy.useproxy value=true

Turns on HTTP web proxy usage.
