geo-bounds-use-source Command

Sets the geometry boundaries to correspond to a specific map source.

Usage 

geo-bounds-use-source <source index>

Parameters:

  • source index: optional parameter specifying the index of the map source. If not specified, the last used map source will be used.

Examples 

load-source maps\greenland.osm
geo-bounds-use-source

Loads the Greenland map and uses its extents to set the geometry bounds.

geo-bounds-use-source 3

Uses the extents of the map source with the index of 3 to set the geometry bounds.
