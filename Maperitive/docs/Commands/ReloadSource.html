reload-source Command

Reloads the map source from its file.

NOTE: some map sources like the Map decoration and Highlight cannot be reloaded since they are not tied to a particular file on the disk. Generated map sources (for example using download-osm Command) need to be saved on the disk before trying to reload them (well, obviously).

Usage 

 reload-source [<source index>]

Parameters:

  • source index: optional parameter specifying the index of the map source. If not specified, the command will apply to the last used map source.

Examples 

reload-source 3

Reloads a map source under the index 3.

reload-source

Reloads the last used map source.
