save-file-list Command

Saves the list of files generated from a previous file command into a text file.

Usage 

 save-file-list [<list file name>]

Parameters:

  • list file name: specifies the name of the list file (optional).

Examples 

 generate-tiles
 save-file-list file-list.txt

Generates map tiles and then saves the list of tile file names into the file-list.txt file.
