Maperitive 2.3.33 documentation

list-settings Command

Lists Maperitive user settings, their current values and descriptions.

Usage 

 list-settings [filter]

The optional filter parameter allows searching for user settings which contain the specified filter text in their names.

Examples 

 list-settings

Lists all of the available user settings in Maperitive.

 list-settings proxy

Lists all of the user settings that contain proxy text in their names.
