apply-ruleset Command

Applies the currently active ruleset to a map source or the whole map, depending on the parameters.

Usage 

apply-ruleset <source index>

Parameters:

  • source index: optional parameter which is used to apply the ruleset to a specific map source only.

Examples 

use-ruleset alias=wireframe
apply-ruleset

Applies the built-in wireframe ruleset to the whole map.

use-ruleset location=myrules/hiking.mrules
apply-ruleset 2

Applies the a myrules/hiking.mrules ruleset to the second map source in the map.
