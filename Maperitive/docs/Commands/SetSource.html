set-source Command

Hides or shows the map source. When hidden, the map source is still present in the map, but its elements are not drawn.

Usage 

 set-source 
     [index=<source index>]
     [visible=<true/false>]

Parameters:

  • source index: optional parameter specifying the index of the map source. If not specified, the command will apply to the last used map source.
  • visible: specifies whether the map source should be visible on the map or not.

Examples 

set-source index=3 visible=false

Hides the map source under the index 3.

set-source visible=false

Hides the last used map source.
