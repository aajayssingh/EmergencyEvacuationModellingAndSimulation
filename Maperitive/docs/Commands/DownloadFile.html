download-file Command

Downloads a file from Web and saves it on the disk.

Usage 

 download-file
	url=<download url>
	[file=<output file>]
	[timeout=<download timeout>]

Parameters:

  • url: download URL. You need to specify the URL in the full format (including the http:// part).
  • file: specifies the output file name. If not specified, the command will use the file name extracted from the download URL.
  • timeout: specifies the maximum period to wait for the download server to respond before the command returns an error. Timeout is specified in the D.hh:mm:ss format (D is optional). If not specified, one minute will be used.

Examples 

download-file url=http://download.geofabrik.de/osm/europe/montenegro.osm.bz2

Downloads montenegro.osm.bz2 file from Geofabrik's download server and saves it to the disk.

download-file url=http://download.geofabrik.de/osm/europe/montenegro.osm.bz2 file=test.osm.bz2

Downloads montenegro.osm.bz2 file from Geofabrik's download server and saves it to the disk as test.osm.bz2.

download-file url=http://download.geofabrik.de/osm/europe/montenegro.osm.bz2 timeout=2:0:0

Downloads montenegro.osm.bz2 file from Geofabrik's download server using two hours of timeout.
