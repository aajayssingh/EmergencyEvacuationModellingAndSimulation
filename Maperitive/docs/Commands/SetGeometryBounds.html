set-geo-bounds Command

Sets the geometry boundaries to a specific area.

Usage 

set-geo-bounds <bounds>

Parameters:

  • bounds: optionally specifies the exact geometry bounds in the minlng,minlat,maxlng,maxlat format. If this parameter is not specified, the current map view bounds will be used.

Examples 

set-geo-bounds 20,30,50,40

Sets the geometry bounds to the specific area.

set-geo-bounds

Sets the geometry bounds to the current map view bounds.
