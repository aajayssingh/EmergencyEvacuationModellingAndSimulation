line-offset-sides

Specifies the direction(s) the line should be offset. This property can have following values:

  • left : the line should be offset to the left
  • right : the line should be offset to the right
  • both : the line should be offset on both sides

Example:

define
    line-offset : 12.9:0;13:5
    line-offset-sides : right
draw : line

draws the line offset 5 pixels to the right (on zoom level 13 and higher).
