font-weight

Specifies the weight of the font to be used to render text. The property can have one of these two values:

  • normal: the text will have a normal weight (non-emphasized). This is the default value.
  • bold: the text will be bold.

Example:

define
	font-weight : bold

specifies that the font will be bold.
