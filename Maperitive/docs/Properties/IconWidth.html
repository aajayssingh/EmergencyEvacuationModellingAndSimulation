icon-width

A zoomable value which specifies the width of the icon in pixels.

Example:

define
	icon-width : 8:10;14:20

specifies that the icons will have width of 10 pixels up to the zoom level of 8 and then grow to the width of 20 pixels at the zoom level of 14 (and above).
