font-family

Specifies the font family to use for rendering text. The font family has to be available on your computer.

Useful links:

  • For a list of popular font families on Windows and other platforms, please visit http://www.codestyle.org/css/font-family/sampler-CombinedResults.shtml
  • TypeBrewer: A Map Design Help Tool for Selecting Typography

Example:

define
	font-family : Times New Roman

specifies that the font to use is Times New Roman.
