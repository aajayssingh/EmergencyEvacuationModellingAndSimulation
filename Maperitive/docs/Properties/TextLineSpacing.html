text-line-spacing

Specifies the line spacing to use for the text. Line spacing is applied when the text label is word-wrapped. Line spacing is specified as a ratio: a default value of 1 means that the text symbol will use the standard line spacing for the font used. Example:

define
	text-line-spacing : 1.5

specifies that the text will be presented in 50% wider line spacing.
