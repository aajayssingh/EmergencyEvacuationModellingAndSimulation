fill-hatch

Specifies the hatch style to use for the fill.

Possible values are:

  • solid : this is the default hatch style
  • horizontal
  • vertical
  • nwse
  • swne
  • cross
  • diagonalcross

Fill hatches (all except the solid) use two colors. The first color is specified using the fill-color and the second one using the fill-hatch-color. The second color is also affected by the fill-hatch-opacity.
