font-size

A zoomable value which specifies the vertical size of the text to be rendered. The size is specified using the em size of the font.

Example:

define
	font-size : 8:10;14:20

specifies that the font will be of em size of 10 up to the zoom level of 8 and then grow to size 20 at the zoom level of 14 (and above).
