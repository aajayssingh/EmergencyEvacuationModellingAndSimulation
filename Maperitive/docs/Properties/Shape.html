shape

This property specifies the type of the shape or shield to draw.

Possible values are:

  • circle. This is the default value.
  • square
  • triangle
  • diamond
  • custom: tells Maperitive to render a custom shape defined using shape-def property.

Example:

shape-mode: square

specifies that the square will be drawn.
