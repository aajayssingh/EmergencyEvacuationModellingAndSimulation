shape-orientation

This property specifies how the shapes or icons will orient themselves in relation to the feature they are representing. This property has an effect on linear features only.

Possible values are:

  • fixed: the shape/icon orientation will be fixed regardless of the line direction. This is the default value.
  • flow: the shape/icon will orient (rotate) itself in the direction of the line.

Example:

define
	shape-orientation: flow

specifies that the flow orientation will be used.
